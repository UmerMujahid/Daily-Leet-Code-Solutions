# LeetCode Solutions
 
A collection of solutions to various LeetCode problems, organized for easy reference and learning.
 
## 📖 About
 
This repository contains my solutions to LeetCode problems, written in clear and well-commented code. It's meant to serve as both a personal practice log and a resource for anyone looking to deepen their understanding of algorithms and data structures.
 
## 🚀 How to Use
 
1. Browse the repository to find the solution for a specific problem.
2. Files are typically named after the problem (e.g., `two-sum.py`, `merge-intervals.cpp`).
3. Open any file to view the approach, code, and (where available) a brief explanation of the logic and complexity.
## 🌟 Quick Tips
 
- **Navigate:** Use the file browser or search to find a specific problem by name or number.
- **Learn:** Read through the comments to understand the reasoning behind each approach.
- **Contribute:** Found a better approach or a bug? Improvements are always welcome.
